# Módulo de Análisis y Ejecución (Backend Lógico)

Esta carpeta es el contenedor para el núcleo del lenguaje Algoritmia.

**Responsabilidad del Integrante [Nombre de tu compañero]:**
Aquí deben ubicarse los archivos generados por ANTLR4 y la lógica de ejecución:

- `AlgoritmiaLexer.py` & `AlgoritmiaLexer.tokens`
- `AlgoritmiaParser.py` & `AlgoritmiaParser.tokens`
- `AlgoritmiaListener.py` & `AlgoritmiaVisitor.py`
- `Executor.py` (Lógica de visita y generación multimedia)
- `main.py` (Punto de entrada de consola)

> El archivo `__init__.py` es necesario para que Python reconozca este directorio como un paquete.
